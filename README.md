# User Scripts Collection

A collection of useful userscripts to enhance your web browsing experience. These scripts are designed to work with browser extensions like [Tampermonkey](https://www.tampermonkey.net/), [Greasemonkey](https://www.greasemonkey.org/), or [Violentmonkey](https://violentmonkey.github.io/).

---

## 📜 Included Scripts

### 🚀 [Jump to Top/Bottom with Hotkeys](./jump-top-bottom.js)
Adds convenient buttons and keyboard shortcuts to quickly navigate through long pages.

*   **Buttons:** Floating transparent buttons on the right side of the screen.
*   **Keyboard Shortcuts:**
    *   `w` / `W`: Jump to Top
    *   `s` / `S`: Jump to Bottom
    *   `j` / `J`: Scroll down slightly (40px)
    *   `n` / `N`: Scroll down (150px)
    *   `d` / `D`: Scroll down half a page
    *   `k` / `K`: Scroll up slightly (40px)
    *   `m` / `M`: Scroll up (150px)
    *   `a` / `A`: Scroll up half a page

### ☀️ [Light Force](./light-force.js)
Forces dark-themed websites into a clean light mode. Perfect for those who prefer light interfaces even on sites that don't natively support them.

*   **Intelligent Transformation:** Only affects dark-themed sites or elements, leaving naturally light sites alone.
*   **Toggleable:** Includes a menu command in your userscript manager to enable or disable the "Light Force."
*   **Dynamic:** Monitors page changes to ensure new content stays in light mode.

### 🔗 [QR Code Generator Icon](./qr-code.js)
Easily generate a QR code for the current URL to quickly share or open the page on your mobile device.

*   **Simple Access:** A small "🔗" icon stays pinned to the bottom-left corner.
*   **Instant QR:** Clicking the icon displays the QR code.
*   **Easy Dismiss:** Just click the QR code image to hide it.

## 🛠 Installation

1.  **Install a Userscript Manager:**
    *   [Tampermonkey](https://www.tampermonkey.net/) (Recommended)
    *   [Violentmonkey](https://violentmonkey.github.io/)
2.  **Add a Script:**
    *   Open the `.js` file you want from this repository.
    *   Click on the **"Raw"** button to view the raw source code.
    *   Your userscript manager should automatically detect it and ask if you want to install it.
    *   Alternatively, copy the code and "Create a new script" in your manager's dashboard.
